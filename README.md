# Bracelete Aryoli

Bracelete para casais à distância. O usuário aperta um botão e um alerta é enviado ao bracelete pareado, que vibra em resposta.

## Funcionalidades essenciais

- Conexão BLE entre bracelete e aplicativo móvel
- Vibração do bracelete em caso de recebimento de alerta do aplicativo móvel
- Botão no bracelete para envio de alerta ao aplicativo móvel
- Comunicação entre aplicativo móvel e servidor
